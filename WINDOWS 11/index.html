Windows 11 Clone Project

Project Title: Windows 11 Clone

Description: A replica of the Windows 11 operating system, built to demonstrate the ability to mimic the UI/UX and functionality of Windows 11 using HTML, CSS, and JavaScript. The project showcases a simulated environment of the Windows 11 desktop, incorporating key features like the Start Menu, Taskbar, and basic system settings.

Key Features:

  • Interactive Start Menu
  • Taskbar with dynamic icons
  • Window management (minimize, maximize, close)
  • Basic system settings window
  • Hover and click effects for interactivity

Technologies Used:

HTML, CSS, JavaScript
